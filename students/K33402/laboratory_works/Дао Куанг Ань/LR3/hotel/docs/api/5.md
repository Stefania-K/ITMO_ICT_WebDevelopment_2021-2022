# Get ROOM in specific FLOOR

Get number of rooms in specific floor

**URL** : `/api/5/?floor=<floor>`

**Method** : `GET`

**Auth required** : YES

**Permissions required** : None

**Data constraints** : `{}`

## Success Responses

**Code** : `200 OK`

**Content** : `{[]}`

```json
[
    {
        "floor": 1,
        "count": 3
    }
]
```
