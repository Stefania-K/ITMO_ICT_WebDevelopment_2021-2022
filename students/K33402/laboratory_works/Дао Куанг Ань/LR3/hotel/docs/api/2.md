# Get NUMBER of CLIENTS

Get number of clients that come from the same city

**URL** : `/api/2/?city=<city>`

**Method** : `GET`

**Auth required** : YES

**Permissions required** : None

**Data constraints** : `{}`

## Success Responses

**Code** : `200 OK`

**Content** : `{[]}`

```json
[
    {
        "city": "Rockford",
        "count": 3
    }
]
```
